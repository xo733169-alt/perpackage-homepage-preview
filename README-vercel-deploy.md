# Perpackage Brand Pages - Vercel Deploy

This folder is a static Vercel-ready site.

## Deploy target

Upload this folder as the project root:

`peerl-preview/vercel-deploy/perpackage-brand-pages`

## Included pages

- `index.html`
- `company.html`
- `products.html`
- `custom.html`
- `sustainability.html`
- `contact.html`

## Assets

The Notion-selected image assets are included in:

`assets/notion-selected`

## Vercel settings

Use these settings when importing manually:

- Framework Preset: Other
- Build Command: None
- Output Directory: `.`
- Install Command: None
