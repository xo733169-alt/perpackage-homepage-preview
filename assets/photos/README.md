# Perpackage Homepage Photo Library

This folder collects the homepage photo materials supplied for the Perpackage preview site.

## Files

- `perpackage-brand-package-set.jpg`: brand package set for main and company hero sections.
- `perpackage-estimate-desk.jpg`: estimate desk and quote materials for contact and quote sections.
- `perpackage-eco-package-set.jpg`: eco-friendly package set for sustainability and material guidance.
- `perpackage-design-process.jpg`: dieline-to-finished-box process image for custom production.
- `perpackage-blue-detail.jpg`: close-up navy/white package detail for product and finishing sections.

## Current Usage

- `index.html`: main hero, custom service visual, industry card, quote preparation section.
- `company.html`: company hero.
- `products.html`: product hero, material/spec image, finishing detail image.
- `custom.html`: custom production hero and brief section image.
- `sustainability.html`: sustainability hero.
- `contact.html`: contact hero.
